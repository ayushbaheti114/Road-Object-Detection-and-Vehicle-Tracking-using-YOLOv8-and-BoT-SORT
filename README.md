# Road Object Detection and Vehicle Tracking using YOLOv8 & BoT-SORT

This project is an AI-powered traffic surveillance system that performs real-time vehicle detection and multi-object tracking using the YOLOv8 object detection model and BoT-SORT tracking algorithm.

The system identifies vehicles from road surveillance videos, assigns unique tracking IDs, displays confidence scores, and provides real-time traffic analysis including vehicle count, FPS, and traffic density.

# Features

- Real-time Car Detection using YOLOv8
- Multi-Object Tracking using BoT-SORT
- Unique Vehicle IDs
- Confidence Score Display
- Real-time Vehicle Count
- FPS Monitoring
- Traffic Density Detection
- Output Video Generation
- OpenCV Visualization

# Technologies Used

- BoT-SORT
- NumPy
